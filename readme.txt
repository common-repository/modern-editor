Modern user interface for edit screens of posts and pages. Never loose your focus again when you save/publish a post. Look ma no refresh!

== Description ==
Modern editor helps you to edit posts/pages with ligthning fast. You will never loose your focus again when you update/publish a post.

Features
+ Save with ajax without refresh
+ 3X faster than classic editor when saving/updating
+ Saving cmd+s or ctrl+s without pain
+ ACF support
+ New fixed side inspector design for meta boxes
+ Revision meta box design improvements

== Installation ==
1) Install Plugin
2) Enable your post types under \"Settings > Modern Editor\"
4) Enjoy!
